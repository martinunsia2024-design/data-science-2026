## Deskripsi Portofolio
Repository ini berisi kumpulan berkas interaktif Jupyter Notebook (.ipynb) yang merangkum kurikulum dasar keilmuan data sains. Alur pengerjaan di dalam folder ini mengikuti standar metodologi industri data mining, yaitu **CRISP-DM** (Cross-Industry Standard Process for Data Mining). Perjalanan pengerjaan dimulai dari pemahaman konsep pemrograman dasar Python, manipulasi matriks numerik tinggi dengan NumPy, pengolahan tabel heterogen dengan Pandas, taktik pembersihan data kotor (Data Cleaning), komunikasi visual (Storytelling Data), transformasi pra-pemodelan (Data Preprocessing), hingga ditutup dengan implementasi algoritma kecerdasan buatan (Machine Learning Regresi Linier).

## Ekosistem Pustaka (Tools & Library)
Proyek dan seluruh kode program di dalam portofolio ini dibangun menggunakan ekosistem bahasa pemrograman **Python 3** dengan memanfaatkan pustaka utama sebagai berikut:
* **Pandas** & **NumPy**: Untuk pengolahan struktur data tabel terkomputasi tinggi.
* **Matplotlib** & **Seaborn**: Untuk pembuatan komponen grafis visualisasi data.
* **Scikit-Learn (Sklearn)**: Untuk kebutuhan preprocessing data (imputasi, scaling, encoding) serta pemodelan model machine learning regresi.
* **SciPy**: Untuk analisis perhitungan fungsi statistika lanjut.

## Cara Menjalankan Notebook
1. Salin alamat tautan repository ini: `https://github.com/martinunsia2024-design/data-science-2026`
2. Buka platform [Google Colab](https://colab.research.google.com/) di browser internet Anda.
3. Pilih menu **File** -> **Open Notebook** -> Klik tab **GitHub**.
4. Tempelkan URL tautan repository ini, tekan Enter, lalu pilih salah satu file pertemuan yang ingin dijalankan.
5. Jalankan seluruh baris kode secara berurutan dengan menekan tombol perintah **Runtime** -> **Run All**.

## Kesimpulan Umum Perjalanan Belajar (Sesi 1-7)
Melalui rangkaian proses pembelajaran dan praktikum mandiri sepanjang 7 sesi ini, saya menyimpulkan bahwa keilmuan Data Science bukan sekadar keahlian mengetik baris kode pemrograman atau penguasaan terhadap pemakaian alat-alat (*tools*) canggih semata. Data Science sejatinya adalah sebuah kerangka berpikir terstruktur, logis, dan ilmiah untuk menggali nilai atau pengetahuan tersembunyi dari balik sebaran data mentah demi memecahkan masalah nyata di dunia industri. Tahapan pembersihan (*data cleaning*) dan persiapan data (*preprocessing*) diakui merupakan kunci penentu terbesar (porsi kerja 60-80%) yang menjamin kualitas serta kesuksesan akurasi sebuah model kecerdasan buatan, jauh sebelum algoritma Machine Learning tersebut diuji coba ke sistem produksi.
